# phonemmerce_backend
## Technologies
  * Node.js
  * Express
  * Mongodb
  * Mongoose
  * Mocha/Chai
  * Eslint
  * Babel

## Installation
 * Ensure you have node 12.x.x installed.
 
 * Install node modules with the command
 
   * npm install
   
 * Start the API server with command
 
   * npm start
   
 * Check API index with Postman
 
   * http://localhost:3004/api/v1
   
 * Run test with
 
   * npm test
